# **CS251 Data Structures Class Notes**
## **Table of Contents (Time Complexity here is in average case)**
* AVL Tree	
	- Search O(log n)
	- Insert O(log n)
	- Remove O(log n)
* Binary Search Dictionary
	- Search O(log n)
	- Insert O(log n)
	- Remove O(log n)
* Binary Search Tree
	- Search O(log n)
	- Insert O(log n)
	- Remove O(log n)
* Double Linked List
	- Search O(n)
	- Insert O(1)
	- Remove O(1)
* HashTable
	- Search O(1)
	- Insert O(1)
	- Remove O(1)
* Heap
	- Insertion O(log(n))
	- HeapSort O(n log(n))
* Queque
	- Search O(n)
	- Insert O(1)
	- Remove O(1)
* Quick Sort 
	- O(n log(n))
* Merge Sort 
	- O(n log(n))
* Radix Sort 
	- O(n k) k is the upper range of the elements
* Bucket Sort
	- O(n + k) k is the number of buckets
* Rabin-Karp Algorithm
	- O(n + m)
